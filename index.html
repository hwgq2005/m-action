Usage

调用方式：
var action = Action({
	id:'action',
	summary:'这是一段短文',
	actions:[{
		name:'确定',
		callback:function(){

		}
	},{
		name:'返回上一步',
		callback:function(){
			
		}
	}],
	complete:function(){
		// do something ...				
	},

	cancel:function(){
		// do something ...
	}
})
属性 备注说明
id string，例如："action"
summary string，描述
actions array，数组值是个对象，内含name和callback
backdrop boolean值，true或者false
cancelButton boolean值，true或者false
cancelText string，取消按钮名称
complete function(){}，打开弹框后回调
cancel function(){}，取消事件回调
